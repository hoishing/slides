# 分宮制

<https://bit.ly/3SqE8Pc>

- Placidus 普拉西德: time used for each deg from IC to Asc
- Porphyry 波菲利: 上升 + 天頂 / 3
- whole sign 整宮制: 上升星座 = 1 宮
- equal 等宮制: 上升點 = 1 宮頭
